Using handles instead of direct calls/writes

Global $hDLL=DllOpen("some.dll")
Global $hFILE=FileOpen("some.file")

For $i=1 to 1000
  DllCall($hDLL)
  FileWrite($hFILE, "something")
Next

FileClose($hFILE)
DllClose($hDLL)

is always a bit faster than

For $i=1 to 1000
  DllCall("some.dll")
  FileWrite("some.file", "something")
Next
