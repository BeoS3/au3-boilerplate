Sciter SDK, table of content

  • doc/ - HTML documentation. Use your favorite browser or '?' button on Sciter toolbar.
  • lib/ - reusable componets, scriptable behaviors and modules.
  • samples/
    • addins/ - demos of external binary behaviors (add-ins);
    • html-and-css/ - HTML and CSS samples. This is copy of /html_samples/ folder of htmlayout SDK
    • tests/ - various basic tests that can be used as samples too.
  • sciter.exe - The Sciter application. Standalone executable - does not require any other DLLs or modules. Does not require installation.
  • sciter-x.dll -The Sciter packaged as plain win32 dll.
