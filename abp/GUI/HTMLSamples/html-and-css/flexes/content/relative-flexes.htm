relative flexes in flow:horizontal

All list items here have height:* and so they span the whole height of the row. If height of such row element is not set (as in this sample) then it will be equal to tallest element in the row.

Case 1, different base lengths, same flex value:

  • width:fx(1,200px)
  • width:fx(1,400px)

Case 2, same base length - 200px, different flex values:

  • width:fx(1,200px)
  • width:fx(2,200px)

Case 3, same flex value, base lengths are using max-intrinsic widths:

  • width:fx(1,max-intrinsic)
  • width:fx(1,max-intrinsic), more content added
