The element should contain "Passed!" text inside:

Notation of the @include statement in CSS: 
@include mime-type url [media list];
Where:
  • mime-type - string, mime type, e.g. "text/tiscript";
  • url - url(), url of script file to include, e.g. url(script-for-include.tis);
  • media list - whitespace separated list of nmtokens - media types.
