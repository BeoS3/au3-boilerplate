Sudoku game.

Menu:

- new: 1st toolbar menu. Start a new game.
- load: 2nd toolbar menu. Load saved game from the storage.
- save: 3rd toolbar menu. Save game into the storage.
- check: 4th toolbar menu. Check the game. This option could be used at any stage of the game. Errors will be highlighted.
 
To enter values into table cells, use left mouse click and select one of the values from the popup.
