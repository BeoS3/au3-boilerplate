Notes:

In CSS Mobile Sciter supports following media types: screen, handheld, hires.

So if you need to design CSS screen resolution aware then use:

@media hires
{  
 #myimg { foreground-image: url(...) }
}

Remeber that you can use same CSS files on desktop and mobile. When you need to tune up styles for handheld use something like this:

@media handheld
{  
 H1 { font-size:16pt; }
 #myimg { foreground-image: url(...) }
 @media hires
 {  
   #myimg { foreground-image: url(...) }
 }
}
